# Auto Zapret + Discord Low Priority

Автоматический `.bat`-скрипт для работы с zapret-конфигами и запуска Discord с минимальной нагрузкой на систему.

## 🚀 Возможности

* 🔍 Автоматически перебирает `general*.bat`, находит рабочий и запускает Discord при успошном обх*де
* 💾 Запоминает рабочую стратегию (в `working.txt`)
* ⚡ При следующем запуске сразу использует её
* 🔄 Если перестала работать — автоматически ищет новую
* 🎯 Запускает Discord
* 🧊 Устанавливает **самый низкий приоритет (Idle)** для всех процессов Discord, чтобы не было лишней нагрузки в фоне, например для игры в CS2
* 🧼 Работает без лишних окон и без ручного ввода, за исключением того, что до полного запуска Discord будет висеть пустое окно cmd, которое само закроется.

---

## 📦 Требования

* Windows 10 / 11
* Наличие `curl` (встроен в современные версии Windows)
* zapret-discord-youtube

---

## ⚙️ Установка

1. Скачай или клонируй репозиторий zapret
2. Помести `select.bat` в папку, где находятся `general*.bat`
3. Запусти файл

---

## 🧠 Как это работает

1. Проверяет, есть ли сохранённая рабочая стратегия
2. Если есть:

   * запускает её
   * проверяет доступ к Discord
3. Если не работает:

   * перебирает все `general*.bat`
4. После нахождения рабочей:

   * сохраняет её
   * запускает Discord
   * понижает приоритет всех процессов

---

## 📁 Структура

```
/zapret-folder
│
├── general1.bat
├── general2.bat
├── ...
├── auto_zapret.bat
└── working.txt (создаётся автоматически)
```

---

## ⚠️ Важно

* Одновременно может работать только один конфиг (ограничение WinDivert)
* Антивирус может мешать работе
* Иногда требуется несколько секунд для корректной проверки

## 🔧 Настройка

### Изменить сайт для проверки

В коде:

```
set "TEST_URL=https://discord.com"
```

Можно заменить на:

* https://youtube.com
* или любой другой
